\# IE 221 Probability Project: SLLN \& CLT Verification



\## Project Title and Description

This project aims to experimentally verify the Strong Law of Large Numbers (SLLN) and the Central Limit Theorem (CLT) using simulation. It also includes a Monte Carlo estimation of Pi as part of the IE 221 course requirements.
